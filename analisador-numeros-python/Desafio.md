# Desafio DIO - Algoritmos em Python
Este repositório foi criado como parte de um desafio da plataforma DIO, com o objetivo de praticar lógica de programação em Python, organização de código e documentação de um projeto no GitHub.

## Sobre o Projeto
O projeto consiste em um analisador simples de números inteiros.

A aplicação recebe uma lista de números digitados pelo usuário e retorna algumas informações básicas sobre esses valores, como soma, média, maior número, menor número e quantidade de números pares e ímpares.

## Funcionalidades
- Receber números inteiros informados pelo usuário
- Calcular a soma dos números
- Calcular a média dos números
- Identificar o maior número
- Identificar o menor número
- Contar quantos números são pares
- Contar quantos números são ímpares
- Tratar entradas inválidas de forma simples

## Tecnologias Utilizadas
- Python 3
- Git
- GitHub
